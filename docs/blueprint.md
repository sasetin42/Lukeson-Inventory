# **App Name**: StockPilot

## Core Features:

- Dashboard Overview: Real-time dashboard providing a snapshot of current stock levels and recent transactions.
- Stock Tracking: Track stock levels for each product and categorize items.
- Automated Invoice Generation: Generate invoices automatically when sales transactions are completed.
- Supplier Profiles: Maintain detailed profiles for each supplier, including contact details and contract terms.
- Suggested Actions: Tool: Generate suggested actions to a company based on the state of their stock management.
- Tag extraction: Analyze user descriptions of products to extract useful tags.

## Style Guidelines:

- Primary color: Moderate blue (#5D9CEC) for trust and efficiency.
- Background color: Very light blue (#F0F8FF).
- Accent color: Soft purple (#B19CD9) for highlighting key actions.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look.
- Use consistent, clear icons to represent different inventory items and actions.
- Implement a clean, intuitive layout for easy navigation.
- Subtle animations for transitions and feedback to improve user experience.
